# Workflow Reference

## Suggested fused path

1. `/opsx:explore` -> Intake
2. `/opsx:new` -> change creation + Plan start
3. `/opsx:ff` -> spec framing / plan refinement
4. `/opsx:apply` -> Dispatch + Execute
5. `/opsx:verify` -> Review + QA
6. `/opsx:archive` -> Deliver + Archive + Retro

## Feedback loops

- verify -> apply
- verify -> ff
- verify -> new

Use the narrowest possible artifact bundle when handing off between stages.
