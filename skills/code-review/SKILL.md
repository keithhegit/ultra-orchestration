# Code Review

Review implementation results with a production mindset.

## Check

- Spec compliance
- Correctness and regression risk
- Evidence quality
- Missing tests or verification gaps
- Unintended scope expansion

Use review lenses from [`../ultra-orchestrator/references/review-lenses.md`](../ultra-orchestrator/references/review-lenses.md).

## Output

Lead with findings. If there are no findings, state that explicitly and note remaining verification gaps.
